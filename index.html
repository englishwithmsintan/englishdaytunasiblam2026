🎙️ Experience Interview Stations

Setiap nomor-nomor di bawah ini merupakan pos pertanyaan. Dengan pasanganmu,tanyakan pertanyaan-pertanyaan berikut ini secara bergantian!

1
"Have you ever visited a beach?"
If YES →

"Where was it?"

If NO →

"Would you like to try?"

2
"Have you ever tried a new sport?"
If YES →

"What sport was it?"

If NO →

"Would you like to try?"

3
"Have you ever performed on stage?"
If YES →

"What did you perform?"

If NO →

"Would you like to try?"

4
"Have you ever cooked something?"
If YES →

"What did you cook?"

If NO →

"Would you like to try?"
